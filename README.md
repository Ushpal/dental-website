# Hamro Dental Clinic Website

A modern React + Vite landing page for Hamro Dental Clinic, built with TypeScript and optimized for dental tourism, pediatric care, and professional dental services in Kathmandu.

## About

This site showcases a premium dental clinic offering:
- Pediatric and family dentistry
- Prosthodontics, periodontics, orthodontics, and oral surgery
- Dental tourism packages for Non-Resident Nepalis and international visitors
- High-standard sterilization and patient comfort
- Expert clinical team profiles and patient testimonials

## Built With

- React
- TypeScript
- Vite
- Tailwind CSS (via `@tailwindcss/vite`)
- Lucide icons
- Motion animation utilities

## Project Structure

- `index.html` - entry HTML for the Vite app
- `src/main.tsx` - React app bootstrap
- `src/App.tsx` - main landing page layout and content
- `src/index.css` - global styles
- `package.json` - project scripts and dependencies
- `vite.config.ts` - Vite configuration

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the site in your browser at:

```text
http://localhost:3000/
```

## Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
